Projeto "A Casa Assombrada"

Descrição do projeto

O segundo projeto, trata-se do jogo “A Casa Assombrada”, um jogo que tem como temática o halloween, a finalidade desse jogo é entreter pessoas através de um sistema de jogo de 7 erros, itens foram editados e escondidos dentro de uma tela de modo que os jogadores devem achar esses 7 elementos, mas no meio deles há outros elementos que também foram animados, mas que não fazem parte da contagem, esse efeito foi criado para fornecer uma melhor interação com o usuário. O jogo possui diversos “cômodos” que são sessões do jogo onde é alterado o layout da tela e novos elementos devem ser achados.
